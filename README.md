### Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

5. Clonar el archivo `.env.template` y renombrar la copia a `.env`

6. Llenar la variables de entorno definidas en el `.env`

7. Ejecutar la aplicación de dev:

```
yarn start
```
